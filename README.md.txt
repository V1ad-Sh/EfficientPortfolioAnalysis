# Efficient Portfolio Analysis

This repository contains Python code used for the analysis and construction of efficient portfolios for US technology sector stocks from the NASDAQ-100 index. The data spans from March 1, 2021, to March 1, 2024.

## Features
- Fetch historical stock data using Yahoo Finance API.
- Calculate expected returns, covariance matrices, and risk-free rates.
- Perform efficient frontier optimization.
- Plot and analyze portfolio performance.

## Usage
To execute the analysis, run: python main.py
